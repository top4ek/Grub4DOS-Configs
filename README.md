Scripts
=======
Some of my scripts

- beep-imperial.sh: beeps imperial march in bg ;)
- gentoolike-bashrc: change PS env to gentoo-like
- keenetic-udhcpd.sh: hack to make zyxel keenetic pass PXE options from build-in dhcpd daemon
